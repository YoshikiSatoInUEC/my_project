株式会社翔泳社より刊行の書籍『HTML&CSS 標準デザイン講座 【HTML5＆CSS3対応】』で利用する学習用サンプルファイルです。
本書中でサンプルファイルを使った学習があるLESSONでは、使用するファイルの場所をLESSON冒頭に掲載していますので、記載に従ってファイルを開き、学習を進めてください。


※なお、サンプルのHTMLファイルとCSSファイルは全て「改行コード」が「LF」になっています（本書019ページMEMO参照）。Windows標準の「メモ帳」で開いてしまうと、改行が反映されずに表示が崩れてしまうので、Windowsをご利用の場合は本書006ページに掲載しているようなテキストエディタを使ってファイルを開くようにしてください。

※本書刊行後に判明した誤記載の訂正や補足情報などは、下記URLの翔泳社刊行物Q&Aページにて公開しています。ご参照ください。
http://www.shoeisha.co.jp/book/qa/
