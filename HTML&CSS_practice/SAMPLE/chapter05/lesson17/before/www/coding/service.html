HTML5+CSS3による次世代標準コーディングから従来型のXHTML+CSSまで柔軟に対応。

従来通りIE6を含むクロスブラウザ対応をお望みのお客様にはこれまで通りXHTML1.0+CSS2.1による標準コーディングを、スマートフォン・タブレット専用サイトをお考えのお客様にはHTML5+CSS3による次世代標準コーディングを、PC向けサイトでありながらより豊かな表現力を持つ最新ブラウザのユーザ向けにリッチなコンテンツを提供したいとお考えのお客様には対応ブラウザと表現方法をご相談させていただきながら最適な構築方法をご提案。過渡期ならではの様々なご要望にも、柔軟に対応させて頂きます。

PC向け（IE6+）
XHTML1.0+CSS2.1
IE6を含む各ブラウザにおける表示・動作の同一性を重視する「クロスブラウザ対応」を基本とし、古いブラウザで利用が難しい技術は原則として使用いたしません。
PC向け（IE9+）
XHTML1.0 / HTML5+CSS3※
環境が整っているユーザー（モダンブラウザ利用者）には積極的によりリッチな見せ方でコンテンツを提供する「プログレッシブエンハンスメント」という考え方に基づいて構築します。
スマホ・タブレット向け
HTML5+ CSS3
古いブラウザとの互換性に悩まされる心配がないため、最新技術であるHTML5・CSS3の機能を最大限に活用した構築を行います。

※PC向けサイトでのCSS3利用は、一部のプロパティに限定されます。

料金表

全画面のデザインデータは不要です。

多くのコーディング会社が、コーディングのために全画面のデザインデータを用意するように求めていますが、コーディングのために何十、何百ページもあるサイトの全ての画面のデザインデータを用意するのは時間・労力の無駄遣いです。
私たちは、同一デザインで展開できるテンプレート画面のみご用意いただければ残りの画面に関してはパーツ制作も含めて弊社で対応させていただいております。
